## ğŸ’» Getting Started <a name="getting-started"></a>

Project Math-Magicians is a Single Page App with React having three sub-pages which includes an Calculator app within the subpage. It also includes quotes realted to Mathematics in one of the subpage. This project also makes use of APIs and the main aim of this project is to get started with the React library.

### Prerequisites

In order to run this project you need:

 Linters (anything that has been setup will be installed)

```sh
 cd project-math-magicians
 npm install
```

### Setup

Clone this repository to your desired folder:
```sh
  cd my-folder
  git clone git@github.com:tan12082001/project-math-magicians.git
```

### Usage

To run the project, if you are using a Code Editor make use of a live server.
If you are working with webpack you can run the local host with the following command.

```sh
  npm start
```

### Deployment

You can deploy this project using:

```sh
  gh-pages
```
